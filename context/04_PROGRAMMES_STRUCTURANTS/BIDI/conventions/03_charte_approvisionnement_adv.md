# CHARTE D’APPROVISIONNEMENT LOCAL
## Programme BIDI  
### Phase Pilote 2026–2027 – Ebolowa

---

# 1. Préambule

Le programme BIDI est une initiative portée par NZUANE dans le cadre du projet territorial EBWA2035.

Il vise à structurer une production agricole locale organisée afin de soutenir en priorité les orphelinats d’Ebolowa, puis les établissements scolaires.

La production est assurée par la plateforme agropastorale ROMBONE.

Le dispositif technique est encadré par le Collège Régional d’Agriculture (CRA).

La présente charte définit les principes de collaboration entre le programme BIDI et les Acteurs de Vente (ADV) des établissements scolaires.

---

# 2. Objectif du dispositif

Le programme BIDI a pour objectif de :

- Sécuriser l’approvisionnement des ADV ;
- Stabiliser progressivement les coûts d’achat ;
- Réduire les pertes liées aux fluctuations du marché ;
- Améliorer la qualité des produits proposés aux élèves ;
- Renforcer l’économie locale.

BIDI ne remplace pas les ADV.  
Il vise à structurer leur approvisionnement et à renforcer leur activité.

---

# 3. Produits concernés

Les produits proposés dans le cadre du programme comprennent, selon saison et disponibilité :

## 3.1 Matières premières agricoles

- Manioc  
- Maïs  
- Haricot  
- Arachide  
- Plantain (selon disponibilité)  
- Ndolé  
- Gombo  
- Tomate  
- Gnetum (Okok/Eru)  
- Piment  
- Autres produits saisonniers  

Cette liste est indicative et évolutive selon les cycles de production.

## 3.2 Produits semi-transformés

Progressivement, selon les capacités de production :

- Farine de maïs  
- Dérivés de manioc  
- Pâte d’arachide  
- Transformations simples locales  

Ces produits visent à réduire le temps de préparation, améliorer la régularité et limiter les pertes.

---

# 4. Organisation des menus

Le programme BIDI propose un système de menus glissants :

- 4 plats structurés disponibles chaque semaine ;
- Rotation partielle toutes les 2 semaines ;
- Adaptation selon saisonnalité et disponibilité réelle.

Les ADV restent responsables :

- De la préparation des plats ;
- De la vente ;
- De la gestion quotidienne de leur activité.

BIDI structure uniquement l’approvisionnement.

---

# 5. Engagement qualité

Les produits BIDI :

- Sont issus d’une production locale organisée (ROMBONE) ;
- Sont encadrés techniquement par le Collège Régional d’Agriculture (CRA) ;
- Font l’objet d’un suivi agronomique structuré ;
- Sont traçables ;
- Respectent les standards d’hygiène.

### Sécurisation de la chaîne d’approvisionnement

Le programme BIDI repose sur une chaîne d’approvisionnement locale identifiée et suivie.

Les produits proposés :

- proviennent de sources connues et localisées ;
- sont produits dans un cadre structuré et contrôlé ;
- permettent une meilleure transparence sur l’origine des denrées.

Cette organisation contribue à réduire les risques liés aux circuits d’approvisionnement informels ou non maîtrisés.

Les ADV peuvent, si elles le souhaitent, visiter le site pilote ROMBONE.

---

# 6. Organisation pratique

Les modalités d’approvisionnement seront définies progressivement avec les ADV pilotes :

- Planning hebdomadaire de disponibilité ;
- Quantités prévisionnelles ;
- Modalités simples de commande ;
- Modalités de paiement adaptées ;
- Ajustement progressif selon les retours terrain.

---

# 7. Phase pilote 2026–2027

La phase pilote concerne :

- Un nombre limité d’établissements volontaires ;
- Un volume progressif ;
- Un dialogue permanent avec les ADV.

L’objectif est d’améliorer progressivement le dispositif sans perturber l’activité existante.

---

# 8. Esprit du partenariat

Le programme BIDI repose sur un principe simple :

> Structurer localement pour renforcer les acteurs existants.

La réussite du dispositif dépend :

- De la confiance ;
- De la transparence ;
- Du dialogue permanent.
